== Description ==

This plug-in to export as HTML particular article.
- Replace the path of the src attribute and href attribute
- Copy to the upload directory

== Installation ==

1. Please download from http://www.7-2.jp/downloads/t72_output_html.zip
2. Unzip the downloaded package and upload into your WordPress plugins directory.
3. Go to plugins list and activate "Output HTML T72".
